Starry

Starry is an old-school space shooter type game that i made in Monogame! Monogame is a more challenging, code-centric framework because it provides lower-level control over the game loop, graphics, input, and other systems, instead of Unity which has more built-in tools.

As you might already see i styled the game like an old school GameBoy game, which meant i had way less room to make the sprites, but i think it still came out good.

When i had first started to use MonoGame, it was a little frustrating due to me being more used to everything being done for me within Unity, so i had some troubles, but using it more and more, it became a good learning experience, making me use more logical solutions, and now using Unity, i have improved alot!

Platform: PC

Technologies: MonoGame, C#
